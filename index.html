一篇博客的来源

一篇博客的步骤
1.在桌面blog文件夹里单击右键，点Git Bash here
2.输入hexo new “你的名称”（双引号必须英文）
3.回车根据提示在blog文件的sourse中找到你建立的test
4.在test里写上你想要写的东西 可以是日记 代码
5.在blog文件目录里 输入cmd 回车
6.在跳出来的框里输入hexo g -d 回车
7.等待几分钟 在 https://dlingfjhmmm.github.io/ 登录网页 就可以看到你发的东西

删除博客的步骤
1.在文件夹删除你想要删除的test
2.在 cmd后 输入 hexo g -d
3.等待几分钟就可以删除了

成绩排序（杨大利审核）
用一维数组存储学号和成绩，然后，按成绩排序输出。

输入格式

输入第一行包括一个整数N(1<=N<=100)，代表学生的个数。
接下来的N行每行包括两个整数p和q，分别代表每个学生的学号和成绩。

输出

按照学生的成绩从小到大进行排序，并将排序后的学生信息打印出来。
如果学生的成绩相同，则按照学号的大小进行从小到大排序。

#include<stdio.h>
struct Student
{
int number;
int score;
};

int main()
{
int n,i,j;
struct Student students[101];
struct Student temp;
while(scanf(“%d”,&n)!=EOF)
{
for(i=0;i<n;i++)
scanf(“%d %d”,&students[i].number,&students[i].score);
for(i=0;i<n-1;i++)
{
for(j=0;j<n-i-1;j++)
{
if(students[j].score>students[j+1].score)
{
temp=students[j];
students[j]=students[j+1];
students[j+1]=temp;
}
else if(students[j].score==students[j+1].score)
{
if(students[j].number>students[j+1].number)
{
temp=students[j];
students[j]=students[j+1];
students[j+1]=temp;
}
}
}
}
printf(“Output:\n”);
for(i=0;i<n;i++)
printf(“%d %d\n”,students[i].number,students[i].score);
exit(0);
}
return 0;
}
