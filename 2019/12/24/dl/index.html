直角三角形（焦健审核）

题目描述

用3点来代表一个三角形，输入三个点的坐标，实现判断此三角形是不是直角三角形，并输出此三角形的周长。

输入格式

每行6个整数分别表示三个点的横纵坐标。

输出

第一行根据是否直角三角形输出Yes或No，第二行输出三角形的周长，保留小数点后两位。如果不是三角形，输出错误信息“It cant be a Triangle！”

#include<stdio.h>
#include<math.h>
int main()
{
int x1,y1,x2,y2,x3,y3;
double a,b,c,sum=0,t1,t2,t3;
printf(“Please input ax,ay bx,by cx,cy:\n”);
scanf(“%d,%d %d,%d %d,%d”,&x1,&y1,&x2,&y2,&x3,&y3);
printf(“Output:\n”);
t1=pow((x1-x2),2)+pow((y1-y2),2);
t2=pow((x1-x3),2)+pow((y1-y3),2);
t3=pow((x2-x3),2)+pow((y2-y3),2);
a=sqrt(t1);
b=sqrt(t2);
c=sqrt(t3);
if(a+b>c&&a+c>b&&b+c>a)
{
if(aa+bb==cc||aa+cc==bb||bb+cc==a*a)
printf(“YES\n”);
else
printf(“NO\n”);
sum=a+b+c;
printf(“%.2lf”,sum);
}
else
printf(“It can’t be a Triangle!”);
return 0;
}
